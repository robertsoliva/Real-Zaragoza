# Real-Zaragoza
This repository tries to cover an analysis on and off the pitch of Spanish team Real Zaragoza

## Structure

- `wiki/` — LLM-maintained wiki: atomic, sourced markdown pages on the club's history and current state.
- `next-actions.md` — backlog of planned data pulls (SofaScore, Transfermarkt, ...) and future ideas.
- `.claude/` — instructions for Claude when working in this repo.
