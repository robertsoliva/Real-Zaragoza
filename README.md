# Real Zaragoza

This project covers everything about Real Zaragoza CF: club history, current situation, ownership, finances, and identity — through to analyzing player and match stats, and predicting future match outcomes (see `next-actions.md`).

- `wiki/` — the LLM-maintained knowledge base on the club. See `wiki/README.md` for the index.
- `next-actions.md` — backlog: data sourcing (SofaScore, Transfermarkt), stats analysis, match prediction.
- `.claude/` — working conventions for Claude in this repo.
