# Template: log entry

Each entry is a single line appended to `wiki/log.md`.

## Format

```
[YYYY-MM-DD]||One-line-summary with evidence
```

## Rules

- One line per change set — multiple edits in a single commit produce one entry.
- Summary must state the substance: what changed and what evidence supports it.
- Do not write vague entries like "updated squad.md" — state what was updated and why.

## Examples

```
[2026-06-26]||Skill infrastructure added: contribute-text, log-entry and wiki-page templates, wiki/index.md, wiki/log.md (initial repo setup)
[2026-06-26]||Francho Serrano departure confirmed in squad.md (source: realzaragoza.com official announcement)
[2026-06-27]||Head coach Ibai Gómez sacking reported in current-situation.md; marked as open item pending official confirmation
```
